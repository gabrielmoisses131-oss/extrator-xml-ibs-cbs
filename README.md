# Auditor Fiscal NFC-e (SEFAZ x ADM x FLEX)

## Como rodar (Windows)
1. Instale o Python 3.10+.
2. No CMD, dentro da pasta do projeto:
   ```bat
   python -m pip install -r requirements.txt
   python -m streamlit run app.py
   ```

## Como rodar (Linux/Mac)
```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```
